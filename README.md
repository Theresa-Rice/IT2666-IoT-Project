---

## Description  
The Smart Bin System is an Internet of Things (IoT) project developed for the IT2666 module. It uses sensors and a microcontroller to detect the fill level of a waste bin and display the status through a connected application. The project demonstrates practical IoT concepts such as sensor integration, data monitoring, and basic automation to reduce unnecessary waste collection and improve efficiency.

### Tech Stack
- Windows 10 or later  
- Grove Pi
- Raspberry Pi
- Visual Studio (for Windows Forms application)  

---

### Installation

1. Download or clone the repository from  
   [IT2666 IoT Project Repository](https://github.com/NIXON-KOH/IT2666-IoT-Project)

2. Extract `Hardware.zip` to access wiring diagrams and hardware script.

3. Open the firmware.

4. Open the Windows Forms project in Visual Studio (if applicable).

5. Modify configuration settings such as COM port or sensor thresholds if needed.

---

### Executing Program  

1. Connect the microcontroller to the computer using a USB cable.  
2. Select the correct board and COM port in Arduino IDE.  
3. Upload the firmware to the microcontroller.  
4. Run the Windows Forms application from Visual Studio.  
5. Observe sensor readings and bin status updates.
